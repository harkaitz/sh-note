# SH-NOTE

Fast note taking tool for the terminal and dmenu.

## Help

note

    Usage: note -e|-v|-l|-d|NOTE
    
    Simple note taking utility.
    
    -v      : Show configuration.
    -e      : Edit notes file.
    -l      : List notes.
    -d      : Delete last note.
    NOTE... : Add note.
